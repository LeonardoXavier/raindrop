Get Involved

Raindrop is a very young experiment, and there are many things to do. Here are some areas that could benefit from your attention:

  • Interface Design
  • Security
  • Back-end Data Processing
  • Front-end Extensions
  • APIs for web applications

If you have suggestions or want to talk with others about Raindrop:

  • Get Satisfaction for user questions, general help and enhancement requests.
  • IRC channel is #raindrop on irc.mozilla.org for real-time chatting
  • raindrop-core google group used by the Raindrop developers. Requires approval to join.
  • Bug System used by Raindrop developers. Please use Get Satisfaction for giving feedback.
